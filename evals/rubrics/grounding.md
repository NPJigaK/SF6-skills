# Grounding Rubric

Answers should make evidence boundaries visible without requiring fixed labels.

## Pass

- Uses the expected answer mode.
- Names the relevant evidence boundary when the user asks for exact current facts.
- Refuses or holds when evidence is insufficient.
- Keeps exact current values out of stable concept answers.

## Fail

- Presents generated references as canonical.
- Uses raw, normalized, or manual-review data as final public evidence.
- Treats observations as accepted strategy knowledge without review.
- Requires legacy bracket labels instead of natural answer-mode communication.
