Alex Patch

Universal Changes

  • Drive Rush Overheads: now universally +1 on Block (-3 normally) with enough block pushback to prevent an immediate Throw or low-hitting light chain (e.g. 2LK~2LP)
    • Only applies to standard overhead command normals that can lead to grounded combos on hit
    • Frame Data changes affect A.K.I., Akuma, Chun-Li, Elena, Ryu, Sagat, Terry
    • Pushback changes affect Blanka, C. Viper, E. Honda, Guile, Jamie, JP, Juri, Kimberly, Lily, Luke, M. Bison, Mai, Marisa, Rashid, Sagat, Terry
  • Drive Rush Screen Freeze: pressing an Up input during the freeze will now buffer a jump after the screen freeze ends
    • Still subject to move priority, like special moves canceling the prejump frames
    • Neutral jumps during the freeze will become forward/back jumps if the left/right direction is held after the Up direction
    • Only applies if the character is in neutral during the screen freeze (will not be buffered on wakeup or during a move's recovery)
  • Wakeup Input Buffer: extended by 3 frames (7f → 10f)
