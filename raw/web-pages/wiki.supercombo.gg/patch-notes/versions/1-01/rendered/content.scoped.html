SummaryRashid Release. Bug fixes with some minor balance changes mainly for Modern controls.
LinksPatch Notes


Rashid Patch

  • Rashid released


Universal Changes

  • Drive Rush Cancel
    • Can now be performed with MP+MK (Classic) or Drive Parry button (Modern), only while holding Neutral or Forward
    • Light normals are now cancelable into DRC even on whiff due to the low risk of these easier inputs
  • Drive Parry
    • Fixed issue where MK would come out if Parry was buffered during a screen freeze and then released before the Parry occurred
  • Install Supers
    • Fixed issue where unintended behavior would occur if hit by an attack on the exact frame the install timer expired
  • Assist Combos (Modern)
    • Fixed issue where Assist Combo inputs were not properly buffered, and where the 2nd attack of an Assist Combo would be performed even with only 1 input
  • OD Special Moves (Modern)
    • Fixed issue where some moves could not be manually input while holding the Assist button.


Key Character Changes

Blanka

  • Charge Special Moves - Modern Controls (Rolling Attack, Aerial Rolling Attack, Vertical Rolling Attack)
    • Fixed issue where charge time was unintentionally 10 frames longer and input was not recognized for the usual 10 frames after the Forward or Up input
    • Charge Time: 50f → 40f; Input Recognition Time: 0f → 10f
  • j.MP
    • Fixed issue where special cancel could not be performed if the move was parried
  • Back Throw (4LPLK)
    • Fixed issue where Punish Counter built the same amount of Super gauge is a standard throw
  • HP Aerial Rolling Attack (j.[4]6HP)
    • Fixed issue where Blanka's recovery changed if the opponent blocked while in Burnout


Cammy

  • HK Cannon Strike (j.214HK)
    • Fixed issue where j.HK would come out instead if input at earliest timing
  • OD Quick Spin Knuckle (214PP)
    • Fixed issue where opponent's attack could come out in the wrong direction if input during the side switch


Chun-Li

  • Charge Special Moves - Modern Controls (Kikoken)
    • Fixed issue where meterless version charge time was unintentionally 5 frames shorter; Charge Time: 45f → 50f
    • Fixed issue where input was not recognized for the usual 10 frames after the Forward input (all versions); Input Recognition Time: 0f → 10f
  • Forward/Back/Air Throw (LPLK/4LPLK/j.LPLK)
    • Fixed issue where Charge timing would be reset during the successful throw animations
  • Serenity Stream (214P)
    • Fixed issue where canceling attacks into 214P~K would result in Hazanshu (214K)


Dee Jay

  • Face Breaker (4HK)
    • Fixed issue where Punish Counter Drive Damage was higher vs. airborne opponents
  • Back Throw (4LPLK)
    • Fixed issue where Punish Counter built the same amount of Super gauge is a standard throw


Dhalsim

  • Agile Kick (1LK)
    • Chain cancel version no longer works with 2LK input (Classic controls only; no change to Modern)
  • OD Yoga Arch (236KK)
    • Fixed issue where the projectile would reappear after the SA3/CA cinematic
  • OD Yoga Comet (j.63214PP)
    • Fixed issue where Dhalsim could no longer perform mid-air moves when parried by the opponent


E. Honda

  • 2LP
    • Fixed issue where the chained version could be kara-canceled on startup
  • SA3/CA (214214P)
    • Fixed issue where E. Honda would gain much more Drive gauge when parried than other characters


Jamie

  • Modern 2L
    • Changed from 2LK → 2LP so that he has access to a 4-frame attack


Juri

  • 1-Stock Enhanced Saihasho / Ankensatsu / Go Ohatsu (236LK/236MK/236HK)
    • Fixed issue where the 1st frame could be kara-canceled into SA3/CA (allowing unintended combo routes with 2 stocks)


Kimberly

  • SA3/CA (236233P)
    • Fixed issue where it could hit cross-up (behind Kimberly)


Lily

  • OD Condor Dive (j.PPP)
    • Fixed issue where Lily could perform this from a back jump by plinking the inputs (j.PP~P)
  • Air SA2 (j.236236K)
    • Fixed issue where Lily could not perform this from a back jump if the inputs where plinked
    • Fixed issue where the opponent would not be KO'd if the 2nd hit connected mid-air


Marisa

  • Scutum (Modern 6S)
    • Fixed issue where after a counter-attack, a Down input would cause Marisa to exit the stance even while holding the S button, making it difficult to perform Enfold (6S~2S)


Ryu

  • Drive Impact
    • Fixed issue where extra combo scaling would sometimes be applied, resulting in lower damage


Zangief

  • Fixed issue where Drive gauge recovery during Burnout was slower than other characters
  • SA1 (236236K)
    • Fixed bug where Zangief was considered airborne during the successful throw animation, resulting in less Drive gauge build than usual
