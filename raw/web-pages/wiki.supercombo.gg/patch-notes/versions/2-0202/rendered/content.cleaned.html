Version2.0202
Game Version2.0202
dateApril 15, 2026
SummaryBug fixes for the game system as well as JP and M. Bison.
LinksPatch Notes


Universal Changes

  • Drive Parry
    • Fixed issue where Drive Rush could not be performed at certain timings after parrying an attack, causing the player to be locked into their parry recovery animation when attempting a punish
  • Input Buffer
    • If the opponent's Drive Rush freeze overlaps with a downed character's wakeup input buffer window, the buffer timing will be reduced from 10f → 7f
    • This change was made due to issues caused by the previous patch (Version 2.02) which increased the wakeup input buffer by 3 frames
  • Modern Controls
    • Fixed issue where after certain airborne moves (e.g. Dhalsim's Yoga Float, Rashid's Wing Stroke), if the opponent activated Drive Rush, the airborne character could not use air normals with the Auto button for a short time


Character Changes

JP

  • Departure: Shadow (214P/PP~214HP)
    • Fixed issue where the startup was 1 frame faster when canceled into from 2LK, 5MP, 2MK, or 5HK


M. Bison

  • Devil Reverse + Psycho Mine ([2]8K~P + Mine)
    • Fixed issue where the Psycho Mine would stay embedded if a Drive Reversal was performed after blocking the 2nd hit
  • Devil Reverse + Psycho Mine ([2]8K/KK~P + Mine) / Somersault Skull Diver ([2]8K/KK~K~P + Mine)
    • Fixed issue where the attack could not hit from behind under some circumstances, causing M. Bison to whiff through the opponent's body
