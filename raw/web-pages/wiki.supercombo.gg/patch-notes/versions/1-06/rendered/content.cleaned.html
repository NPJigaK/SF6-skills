Version1.06
Game Version1.06
dateJune 26, 2024
SummaryM. Bison Release. Minor balance changes and bug fixes.
LinksPatch Notes


M. Bison Patch

  • M. Bison released


Universal Changes

  • Perfect Parry vs. Projectiles
    • No longer resets the player's Dash input, allowing for immediate Perfect Parry ~ Forward Dash with a 6MPMK~6 input
  • Drive Reversal
    • Fixed bug where a trade could cause a KO at low health
  • Drive Parry
    • Fixed issue where moves that cause a screen shake would not trigger it when parried
  • Taunts
    • Modern Controls input changed from L+M+H+SP → L+M+H+SP+DP+Assist
  • Back Rise (Wakeup)
    • Fixed issue where holding 2 buttons before a knockdown occurred could result in an unintentional Back Rise on wakeup


Key Character Changes

A.K.I.

  • 5MP
    • Expanded collision box upward to prevent unintended side switches in juggles


Akuma

  • Tenmaku Blade Kick (j.2MK)
    • Fixed bug where combo scaling was reduced when it hit immediately before landing
  • Viscera Piercer (5MP~MP)
    • 2nd hit can no longer be performed when inputting both MP+MK (to prevent accidental TC when attempting a DR Cancel)
  • Bone Crusher Axe Kick (5MK~HK)
    • 2nd hit can no longer be performed when inputting both HP+HK (to prevent accidental TC when attempting a DI Cancel)
  • Block Recovery of Various Attacks
    • Fixed issue where Akuma could cancel his recovery on Block earlier than intended with certain actions
    • Applies to 6HP~6HP, 214LK, and 214LP~6P
  • Gou Shoryuken (623P)
    • Fixed bug where Drive Gain on Perfect Parry was less than intended
  • Tatsumaki Zanku-Kyaku (214K)
    • Fixed bug where Drive Gain on Perfect Parry was less than intended
  • Zanku Hadoken (j.236P/PP)
    • Hurtbox expanded forward
    • Upper half of hitbox changed to combo-only hitbox (easier to jump over or counter with anti-air specials)
  • Ashura Senku (6KKK/4KKK)
    • Hurtbox expanded during Teleport movement
    • Will no longer completely overlap with a cornered opponent
    • These changes make it harder for a Teleport to avoid attacks when Akuma has not fully passed through the opponent
  • Shun Goku Satsu (CA) (LP~LP~6LK~HP)
    • Fixed issue where the opponent's charge timing would be reset during the cinematic side switch


Blanka

  • 2MK
    • Fixed issue where the extra whiff recovery and expanded hurtbox from Version 1.05 did not apply while SA2 was active
  • Raid Jump (Modern Controls: 2MH~M/H)
    • Fixed issue where Modern Blanka could not perform an Air Throw using the Throw button
  • Rolling Attack / Vertical Rolling Attack ([4]6P / [2]8K)
    • Fixed issue where the opponent would not be pushed back on block while Blanka is rebounding


Dee Jay

  • Threebeat Combo 1 (5LP~MK)
    • 2nd hit can no longer be performed when inputting both MP+MK (to prevent accidental TC when attempting a DR Cancel)
  • MK/HK Jackknife Maximum ([2]8MK/HK)
    • Now throw invuln on frames 5-6


Ed

  • Assist Combo 3
    • Fixed issue where the input buffer did not work if stopped after the initial 5H (5HP) and then restarted again
  • Body Blow Combination (5MP~HP)
    • 2nd hit can no longer be performed when inputting both HP+HK (to prevent accidental TC when attempting a DI Cancel)
  • Hitman Combination 1 (5MK~MK)
    • 2nd hit can no longer be performed when inputting both MP+MK (to prevent accidental TC when attempting a DR Cancel)
  • Low Smash Combination (2HK~HP)
    • 2nd hit can no longer be performed when inputting both HP+HK (to prevent accidental TC when attempting a DI Cancel)


E. Honda

  • OD Sumo Smash ([2]8KK)
    • Now throw invuln on frames 5-13


Jamie

  • SA1 (236236K)
    • Drive Damage on Block reduced by 200
    • Opponent's Drive Gain on Parry reduced by 2000
    • Version 1.05 caused SA1 to gain an extra hit, so these changes restore the values to their previous version


JP

  • Assist Combo 2
    • Now activates SA2 when OD Stribog is blocked or absorbed by armor


Juri

  • Death Crest 1 (5MP~4HP)
    • 2nd hit can no longer be performed when inputting both HP+HK (to prevent accidental TC when attempting a DI Cancel)
  • OD Fuhajin (214KK)
    • Version 1.05 reduced the startup time, but failed to adjust various properties (Fuha Stock timing, projectile clash, proximity guard) to match the faster startup
    • Fuha stock obtained 14f → 12f
    • Projectile clash hitbox appears 12f → 10f
    • Proximity guard hitbox starts 7f → 5f and expanded to match the full range of the attack hitbox


Ken

  • Quick Dash (KK)
    • Now only takes priority over OD kick specials when all 3 Kick buttons are pressed
    • In Version 1.05, any 5KK input would cause Quick Dash to take priority, even after inputting the command for a special move


Kimberly

  • OD Sprint (236KK)
    • Fixed bug that would cause sound effect not to play
  • Bushin Senpukyaku (214K)
    • Added Throw Invuln to the 1st hit, starting on the 2nd active frame for each version
    • 214LK now throw invuln on frames 7-11
    • 214MK now throw invuln on frames 8-12
    • 214HK now throw invuln on frames 9-13
    • 214KK now throw invuln on frames 7-11


Luke

  • Assist Combo 1
    • Input is now accepted during the chain cancel window


Marisa

  • Assist Combo 1
    • Input is now accepted during the chain cancel window


Rashid

  • HP Spinning Mixer (236HP Regular/Enhanced)
    • Now throw invuln on frames 8-10


Zangief

  • 5MK
    • Attack hitbox reverted to its pre-1.05 version, slightly reducing its range
  • Tundra Storm (Modern Controls: 22H)
    • Fixed issue where a previous input leniency adjustment was not applied to Modern controls
