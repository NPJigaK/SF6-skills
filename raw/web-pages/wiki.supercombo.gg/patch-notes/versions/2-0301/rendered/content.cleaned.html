Ingrid Patch

  • Ingrid released


Universal Changes

  • Wakeup Drive Reversal
    • When the invincible startup interacts with an opponent's Throw hitbox, a 12f screen freeze now occurs and refunds 1 Drive bar on hit
    • This does not prevent Burnout from occurring before the meter is gained, but it makes Drive Reversal a more cost-effective defensive tool against throw loops
    • As an unintended consequence, the screen freeze affects Kimberly's spraycan setups, causing the opponent's Wakeup Drive Reversal to be punishable if she attempts a throw
  • Punish Counter Throws
    • The opponent no longer gains double the usual amount of Super Gauge (no change to attacker's meter gain)
    • This makes Punish Counter Throws a more desirable option to finish a round while building meter without also benefiting the opponent
  • Throw Escapes
    • Defender's Drive Gain reduced 10000 → 5000
    • Defender's Super Gain reduced 1000 → 0
  • Super Projectile Clashes
    • Changed the behavior of clashing multi-hit projectile Supers to ensure consistent results
  • Super Art Command Inputs
    • Fixed issue where holding a specific direction would cause some Supers not to be performed
    • Affected Supers:
      • Akuma SA2 (Modern + Manual, while holding Up)
      • C. Viper SA1 (Modern + Manual, while holding Down or Back)
      • Chun-Li SA1/SA2/SA3 (Modern + Manual, while holding Up)
      • Ed SA3 (Classic, while holding Back)
      • Mai SA1/SA2/SA3 (Modern + Manual, while holding Up)
      • Ryu SA2 (Modern + Manual, while holding Up or Down)


Character Changes

Alex

  • Assist Combo 2
    • Fixed issue where input buffer would be disabled after the 2nd hit was blocked when attempting to use another Assist Combo 2
    • Input buffer for the 2nd hit shifted from frame 2 → 7
  • Twisted Drop (Modern Controls: 2L~2H)
    • Fixed issue where 2nd attack would not be performed while holding the Assist button
  • Palm Strikes (Modern Controls: 5M~M)
    • Fixed issue where 2nd attack would not be performed while holding the Assist button
  • Dangerous Armbar (Modern Controls: 2MH~2LM)
    • Fixed issue where immediate scaling was unintentionally applied when using the Throw button instead of LM
  • Hyper Takedown (2PP~2LPLK)
    • 20% Immediate scaling added to match other normal throws


C. Viper

  • Assist Combo 3
    • Fixed issue where the input buffer for the 2nd attack was faster than expected during the 1st hit


Chun-Li

  • Snake Strike (Modern Controls: 214X~5M)
    • Fixed issue where the Punish Counter state was extended by 1 frame when holding Up after being Perfect Parries


Dhalsim

  • Aerial Yoga Teleport Forward (j.6PPP)
    • Fixed issue where Dhalsim could appear behind the opponent facing the wrong direction if both players were cornered


Ed

  • Low Smash Combination (2HK~HP)
    • Startup of 2nd hit reduced 12 → 10
    • Total animation decreased 35 → 33
    • This causes the Target Combo to always be a true blockstring even when delayed (can no longer frame trap the opponent's attempted 2HK punish)
  • OD Psycho Shoot (236PP~6P)
    • Fixed issue where the projectile clash from OD Psycho Spark could remain after shooting the projectile


Elena

  • Starling Beak (Modern Controls: 5M~M)
    • Fixed issue where 2nd attack would not be performed while holding the Assist button


Guile

  • OD Sonic Blade (214PP)
    • Fixed issue where the L+M and M+H versions would not come out with a negative edge (button release) input


Jamie

  • Full Moon Kick (6MK~MK)
    • Increased forward movement and expanded the attack hitbox and proximity guard hitbox of the 2nd hit
      • This makes it easier for the 2nd attack to connect at more ranges (after the patch that increased the pushback of blocked overheads)
    • Reduced pushback on hit/block for the 2nd attack to pull the opponent closer to Jamie


JP

  • SA3 (236236K)
    • Fixed issue where the attack could whiff through opponents that were moving forward quickly


Juri

  • Senkai Kick (6MK)
    • Increased pushback on block to ensure Juri is always out of throw range after DR~6MK


Kimberly

  • SA1 (236236K)
    • Fixed issue where it was difficult to Back Rise after being hit by the 0-can version


Mai

  • Hien Ren Kyaku (Modern Controls: 5L~L~L)

  • Hoshi Kujaku (Modern Controls: 4H~H)
    • Fixed issue where 2nd attack would not be performed while holding the Assist button


Marisa

  • Malleus Breaker (3HP~3HP)
    • Increased forward movement and expanded the attack hitbox and proximity guard hitbox of the 2nd hit (and expanded the hurtbox to match this new range)
    • This fixes the unintended issue where the 2nd hit would whiff after a blocked 3HP as a result of the patch that increased the pushback of blocked overheads


Sagat

    • Fixed issue where inputting 236LPMP while holding down the HP button would result in a MP High Tiger Shot
