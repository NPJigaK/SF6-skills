SummaryC. Viper Release. Minor bug fixes and adjustments for some characters.
LinksPatch Notes


C. Viper Patch

  • C. Viper released


Universal Changes

  • Whiffed Lights > Drive Rush: fixed an issue where some normals would not cancel near the end of the whiff animation but would cancel on hit/block, allowing players to option select the DR cancel. Applies to:
    • A.K.I. 5LK
    • Chun-Li 2LK
    • Ed 2LK
    • Elena 5LK
    • Mai 5LK
    • Sagat 5LK


  • Drive Rush Cancel overlap with Special Move commands: fixed an issue where some normals would cancel into a special move when input with the Parry buttons, rather than canceling into Drive Rush. Applies to:
    • A.K.I. 5LK
    • Akuma 2LP
    • Elena 5LP
    • M. Bison 5LP
    • Marisa 2LP
    • Sagat 2LP, 2MP~HP


  • Super Art affecting opponent's Drive Gauge recovery: fixed an issue where certain Supers with a lock-in animation would stop a burned out opponent's passive Drive Gauge regeneration during the animation. Applies to:
    • Akuma SA2
    • Cammy SA2 (ground/air)
    • E. Honda SA2
    • Elena SA2
    • Kimberly SA2 (ground/air)
    • M. Bison SA2
    • Marisa Held SA1, SA2 (cinematic hit)
    • Sagat SA2 (all variations)


  • Super Art gauge scaling: fixed an issue where some Supers that incur scaling penalties to their meter gain (Chun-Li/Dee Jay SA2) would have follow-up hits scaled even further if landed in a combo with 16 or more hits


Character Changes

Cammy

  • LK Spiral Arrow (236LK) active frame 6~ increased juggle time vs. airborne opponents
    • Ensures that 5HP~HK > 236LK can always juggle into 623K near the corner


Chun-Li

  • OD Tensho Kicks (22KK) expanded collision box upward and expanded final hitbox forward
    • Fixes an issue where the 2nd attack could go in the wrong direction at some distances


Dee Jay

  • OD Waning Moon (214KK~MK) expanded the collision box upward on frames 16-19
    • Primarily intended to fix issue with Modern controls Assist Combo 3 whiffing under opponents


Elena

  • Moon Glider (214P/PP~6P) hitstop on block increased 11 → 13 to match the on-hit version
    • This prevents an option select where follow-ups could be performed only on hit, bypassing the need to hitconfirm it


Guile

  • 2LK fixed issue where Guile had a Punish Counter hurtbox 1f after the end of recovery
    • Fixed issue where a chained 5LP/2LP/2LK would come out when trying to buffer a technique that uses LP or LK buttons


Jamie

  • Freeflow Strikes Drink Lv.4 (236P) expanded collision box upward when first attack hits to prevent later hits from whiffing under the opponent


JP

  • OD Torbalan (236KK) added a combo-only hitbox to prevent whiffing on midair opponents


M. Bison

  • Throw Escape now functions properly when inputs overlap with special moves exclusive to SiRN Bison
  • Head Press ([2]8K~K) no longer bounces back when triggering counter moves (JP's Amnesia, Marisa's Held SA1) making it no longer able to avoid punishment


Manon

  • Modern Control inputs for Renversé (236P) follow-ups (Feint and Grand Fouetté kick) changed to fix input overlap issues when canceling into SA3
    • Renversé Feint changed from 2[S] to 2S~[L]
    • Grand Fouetté changed from 2S~X to 2S~M


Mai

  • MP Ryuuenbu (214MP) no longer applies 20% starter scaling and 2-hit combo scaling when only the 2nd hit connects
  • Flame MP Ryuuenbu (F.214MP) no longer applies 20% starter scaling when only the 2nd hit connects


Ryu

  • OD Denjin Hashogeki (214PP) chip damage corrected from 400 → 200


Terry

  • OD Quick Burn (214LPMP) cancel window into SA2/SA3 now comes out on the 2nd active frame of the 2nd hit
    • Prevents a combo drop that would occur when the 2nd hit's first active frame whiffed


Zangief

  • Double Lariat (Modern Controls: M+H) now comes out properly instead of a Throw when inputting L+M+H during Burnout
