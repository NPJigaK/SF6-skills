# Question: Chun-Liのモダンとクラシックで、フレームデータ上の技数や入力はどう違いますか？

## 短い答え

Chun-LiのModern版は「Classicと同じ技表を、そのまま別入力にしたもの」
ではない。公式Chun-Liフレームデータでは、Classicは78項目、Modernは72項目で、
Modernのほうが6項目少ない。

ただし、公式表の「技名」を完全一致で比べると、Classicにだけある技名は7つ、
Modernにだけある技名は1つになる。これはClassicの `しゃがみ強K（元伝暗殺蹴）`
に対して、Modern側では `元伝暗殺蹴` という別名の項目があるため。

Classicにだけある技名は次の7つ。

| カテゴリー | Classicのみの技名 | Classic入力 |
|---|---|---|
| 通常技 | 立ち中K（外擺脚） | `中K` |
| 通常技 | 立ち強K（前周脚） | `強K` |
| 通常技 | しゃがみ強K（元伝暗殺蹴） | `↓ + 強K` |
| 特殊技 | 水蓮掌 | `↘ + 強P` |
| 特殊技 | 鶴脚落 | `↘ + 強K` |
| 必殺技 | 弱 覇山蹴 | `↓↙← + 弱K` |
| 必殺技 | 強 覇山蹴 | `↓↙← + 強K` |

Modernにだけある技名は次の1つ。

| カテゴリー | Modernのみの技名 | Modern入力 |
|---|---|---|
| 特殊技 | 元伝暗殺蹴 | `↘ + 強` |

入力表示は大半が異なる。公式表の「技名」が同じ71項目のうち、入力表示が
完全に同じなのは `前方ステップ`、`後方ステップ`、`パリィドライブラッシュ`
の3項目だけで、残り68項目はModernとClassicで入力表示が異なる。

つまり、技数と入力でまず見るべき差は「Modernでは表にないClassic専用項目が
いくつかある」「入力は大半がModern用に置き換わる」の2点。

補足すると、共通71技名では、入力とダメージ以外のフレーム系の値はこの公式表
では一致している。一部のModern入力ではダメージが下がる。

## 根拠

この回答は、2026-05-27に取得し、人間レビューでacceptedになったCapcom公式
Chun-Liフレームデータを根拠にしている。Classic/Modernの表を項目単位で比較し、
技名、入力表示、動作フレーム、硬直差、キャンセル、ダメージ、ゲージ増減、
属性、備考の差分を確認した。

Modern側の補足説明には、SPボタンで発動した場合のダメージが80%になるという
説明がある。そのため、Modernの一部項目で `480 (600)` のように低下ダメージと
通常入力時の値が併記されている。

## 比較

カテゴリー別の項目数は次の通り。

| カテゴリー | Classic | Modern | 差 |
|---|---:|---:|---:|
| 通常技 | 19 | 16 | -3 |
| 特殊技 | 16 | 15 | -1 |
| 必殺技 | 25 | 23 | -2 |
| スーパーアーツ | 5 | 5 | 0 |
| 通常投げ | 3 | 3 | 0 |
| 共通システム | 10 | 10 | 0 |
| 合計 | 78 | 72 | -6 |

公式表の「技名」が完全に同じ項目で比較すると、共通項目は71、Classicのみは7、
Modernのみは1。

| 差分の種類 | 件数 | 備考 |
|---|---:|---|
| 入力表示が違う | 68 | 多くのClassic入力が、Modernボタン、SP、AUTO、方向+SPなどに置き換わる。 |
| 入力表示が同じ | 3 | `前方ステップ`, `後方ステップ`, `パリィドライブラッシュ`. |
| ダメージが違う | 15 | Modern側ではSP/簡易入力の低下ダメージが表示され、括弧内に通常入力時の値が併記される場合がある。 |
| 発生 / 持続 / 硬直 / ヒット / ガード / キャンセル / ゲージ / 属性 / 備考が違う | 0 | この時点の公式表では、同じ技名の項目に差分なし。 |

入力差分の例:

- `立ち弱P（手突）`: Classicは `弱P`、Modernは `弱`。
- `強 気功拳`: Classicは `←溜め→ + 強P`、Modernは `←溜め + SP`。
- `行雲流水`: Classicは `↓↙← + P`、Modernは `↓↙← + 全ボタン`。
- `強 天昇脚`: Classicは `↓↓ + 強K`、Modernは `→ + SP`。
- `SA1 気功掌`: Classicは `↓↘→↓↘→ + P`、Modernは `ニュートラルまたは→ + SP / 強`。
- `前方ステップ`: どちらも `→→`。

ダメージ差分がある共通項目:

|---|---:|---:|
| 強 気功拳 | 600 | 480 (600) |
| OD 気功拳 | 800 | 640 (800) |
| 弱 百裂脚 | 800 | 640 (800) |
| OD 百裂脚 | 1000 | 800 (1000) |
| 弱 空中百裂脚 | 900 | 720 (900) |
| OD 空中百裂脚 | 1600 | 1280 (1600) |
| 中 覇山蹴 | 1000 | 800 |
| OD 覇山蹴 | 1200 | 960 |
| 強 天昇脚 | 1200 | 960 (1200) |
| OD 天昇脚 | 1400 | 1120 (1400) |
| SA1 気功掌 | 1700 | 1360 (1700) |
| SA1 空中気功掌 | 2000 | 1600 (2000) |
| SA2 鳳翼扇 | 2000 | 1600 (2000) |
| SA3 蒼天乱華 | 4000 | 3200 (4000) |
| CA 蒼天乱華 | 4500 | 3600 (4500) |

## 注意

- この回答では、公式フレームデータ表に載っている1項目を単位として数えている。
  技分類全体への一般論ではない。
- 「技名が同じ項目」は、公式表の「技名」が完全一致するものだけを数えている。
  `しゃがみ強K（元伝暗殺蹴）` と `元伝暗殺蹴` のように、実質的には対応しそう
  でも技名が違うものは別項目として扱っている。
- 本文中の矢印表記は、読みやすさのために公式表の入力表示を置き換えたもの。
- 「ダメージ以外のフレーム系値に差分なし」は、2026-05-27時点のChun-Li公式表に
  ある同名項目に限った話。Classic/Modernのあらゆる実戦状況が同じだという
  意味ではない。
- この回答は2026-05-27時点の公式表に基づくため、後のCapcom更新で項目数や
  入力が変わる可能性がある。
