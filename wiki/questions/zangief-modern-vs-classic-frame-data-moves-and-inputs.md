# Question: Zangiefのモダンとクラシックで、フレームデータ上の技数や入力はどう違いますか？

## 短い答え

ZangiefのModern版は「Classicと同じ技表を、そのまま別入力にしたもの」
ではない。公式Zangiefフレームデータでは、Classicは72項目、Modernは66項目で、
Modernのほうが6項目少ない。

ただし、公式表の「技名」を完全一致で比べると、Classicにだけある技名は7つ、
Modernにだけある技名は1つになる。これはClassicの `しゃがみ強K（ビッグスタンプ）`
に対して、Modern側では `ビッグスタンプ` という別名の項目があるため。

Classicにだけある技名は次の7つ。

| カテゴリー | Classicのみの技名 | Classic入力 |
|---|---|---|
| 通常技 | 立ち弱P（ブレーンチョップ） | `弱P` |
| 通常技 | 立ち中K（ミドルキック） | `中K` |
| 通常技 | 立ち強K（ロシアンハイキック） | `強K` |
| 通常技 | [連打版]しゃがみ弱K（ショートスタンプ） | `↓ + 弱K` |
| 通常技 | しゃがみ中K（ダイナマイトキック） | `↓ + 中K` |
| 通常技 | しゃがみ強K（ビッグスタンプ） | `↓ + 強K` |
| 特殊技 | スメタナドロップキック | `↘ + 強K` |

Modernにだけある技名は次の1つ。

| カテゴリー | Modernのみの技名 | Modern入力 |
|---|---|---|
| 特殊技 | ビッグスタンプ | `↘ + 強` |

入力表示は大半が異なる。公式表の「技名」が同じ65項目のうち、入力表示が
完全に同じなのは `前方ステップ`、`後方ステップ`、`パリィドライブラッシュ`
の3項目だけで、残り62項目はModernとClassicで入力表示が異なる。

Zangiefでは特に、Classicの一回転・二回転コマンドが、ModernではSP入力や
方向+SP入力に置き換わる項目が目立つ。一方で、弱/中スクリューパイルドライバー
のように、Modernでも一回転入力が表に残っている項目もある。

補足すると、技名が同じ65項目では、入力とダメージ以外のフレーム系の値は
この公式表では一致している。一部のModern入力ではダメージが下がる。

## 根拠

この回答は、2026-05-27に取得し、人間レビューでacceptedになったCapcom公式
Zangiefフレームデータを根拠にしている。Classic/Modernの表を項目単位で比較し、
技名、入力表示、動作フレーム、硬直差、キャンセル、ダメージ、ゲージ増減、
属性、備考の差分を確認した。

Modern側の補足説明には、SPボタンで発動した場合のダメージが80%になるという
説明がある。そのため、Modernの一部項目で `1120 (1400)` のように低下ダメージと
通常入力時の値が併記されている。個別項目のダメージは、公式表に表示されている
値をそのまま扱っている。

## 比較

カテゴリー別の項目数は次の通り。

| カテゴリー | Classic | Modern | 差 |
|---|---:|---:|---:|
| 通常技 | 23 | 17 | -6 |
| 特殊技 | 12 | 12 | 0 |
| 必殺技 | 15 | 15 | 0 |
| スーパーアーツ | 6 | 6 | 0 |
| 通常投げ | 6 | 6 | 0 |
| 共通システム | 10 | 10 | 0 |
| 合計 | 72 | 66 | -6 |

公式表の「技名」が完全に同じ項目で比較すると、共通項目は65、Classicのみは7、
Modernのみは1。

| 差分の種類 | 件数 | 備考 |
|---|---:|---|
| 入力表示が違う | 62 | 多くのClassic入力が、Modernボタン、SP、AUTO、方向+SPなどに置き換わる。 |
| 入力表示が同じ | 3 | `前方ステップ`, `後方ステップ`, `パリィドライブラッシュ`. |
| ダメージが違う | 18 | Modern側ではSP/簡易入力の低下ダメージが表示され、括弧内に通常入力時の値が併記される場合がある。 |
| 発生 / 持続 / 硬直 / ヒット / ガード / キャンセル / ゲージ / 属性 / 備考が違う | 0 | この時点の公式表では、同じ技名の項目に差分なし。 |

入力差分の例:

- `ダブルラリアット`: Classicは `P+P`、Modernは `↓ + SP（通常入力: 中+強）`。
- `弱 スクリューパイルドライバー`: Classicは `近距離で 一回転 + 弱P`、Modernは `近距離で 一回転 + 弱`。
- `強 スクリューパイルドライバー`: Classicは `近距離で 一回転 + 強P`、Modernは `近距離で ニュートラルまたは→ + SP（通常入力: 一回転 + 強）`。
- `SA1 エリアルロシアンスラム`: Classicは `↓↘→↓↘→ + K`、Modernは `ニュートラルまたは→ + SP + 強（通常入力: ↓↘→↓↘→ + 弱）`。
- `SA3 ボリショイストームバスター`: Classicは `近距離で 二回転 + P`、Modernは `近距離で ↓ + SP + 強（通常入力: 二回転 + 強）`。
- `前方ステップ`: どちらも `→→`。

## 注意

- この回答では、公式フレームデータ表に載っている1項目を単位として数えている。
  技分類全体への一般論ではない。
- 「技名が同じ項目」は、公式表の「技名」が完全一致するものだけを数えている。
  `しゃがみ強K（ビッグスタンプ）` と `ビッグスタンプ` のように、実質的には
  対応しそうでも技名が違うものは別項目として扱っている。
- 本文中の矢印表記や一回転/二回転表記は、読みやすさのために公式表の入力表示を
  置き換えたもの。
- 「ダメージ以外のフレーム系値に差分なし」は、2026-05-27時点のZangief公式表に
  ある同名項目に限った話。Classic/Modernのあらゆる実戦状況が同じだという
  意味ではない。
- この回答は2026-05-27時点の公式表に基づくため、後のCapcom更新で項目数や
  入力が変わる可能性がある。
