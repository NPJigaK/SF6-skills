# Question: JPのモダンとクラシックで、フレームデータ上の技数や入力はどう違いますか？

## 短い答え

JPのModern版は「Classicと同じ技表を、そのまま別入力にしたもの」
ではない。公式JPフレームデータでは、Classicは69項目、Modernは65項目で、
Modernのほうが4項目少ない。Modernにだけ存在する技名はない。

Classicにだけある項目は次の4つ。

| カテゴリー | Classicのみの項目 | Classic入力 |
|---|---|---|
| 通常技 | 立ち弱K（ニージニイ・ウダール） | `弱K` |
| 通常技 | しゃがみ中P（ズミヤー） | `↓ + 中P` |
| 通常技 | しゃがみ強K（ジョーキル） | `↓ + 強K` |
| 必殺技 | 強 ストリボーグ | `↓↘→ + 強P` |

入力はほぼ別物になる。共通する65技名のうち、入力表示が完全に同じなのは
`前方ステップ`、`後方ステップ`、`パリィドライブラッシュ` の3項目だけで、
残り62項目はModernとClassicで入力表示が異なる。

つまり、技数と入力でまず見るべき差は「Modernでは表にないClassic専用項目が
4つある」「入力は大半がModern用に置き換わる」の2点。

補足すると、共通65技名では、入力とダメージ以外のフレーム系の値はこの公式表
では一致している。一部のModern入力ではダメージが下がる。

## 根拠

この回答は、2026-05-26に取得したCapcom公式JPフレームデータを根拠にしている。
Classic/Modernの表を項目単位で比較し、技名、入力表示、動作フレーム、硬直差、
キャンセル、ダメージ、ゲージ増減、属性、備考の差分を確認した。

Modern側の補足説明には、SPボタンで発動した場合のダメージが80%になるという
説明がある。そのため、Modernの一部項目で `640 (800)` のように低下ダメージと
通常入力時の値が併記されている。

## 比較

カテゴリー別の項目数は次の通り。

| カテゴリー | Classic | Modern | 差 |
|---|---:|---:|---:|
| 通常技 | 18 | 15 | -3 |
| 特殊技 | 8 | 8 | 0 |
| 必殺技 | 26 | 25 | -1 |
| スーパーアーツ | 4 | 4 | 0 |
| 通常投げ | 3 | 3 | 0 |
| 共通システム | 10 | 10 | 0 |
| 合計 | 69 | 65 | -4 |

公式表の「技名」が同じ項目で比較すると、共通項目は65、Classicのみは4、
Modernのみは0。

共通65技名について、フレームデータの各項目を比較すると次の通り。

| 差分の種類 | 件数 | 備考 |
|---|---:|---|
| 入力表示が違う | 62 | 多くのClassic入力が、Modernボタン、SP、AUTO、方向+SPなどに置き換わる。 |
| 入力表示が同じ | 3 | `前方ステップ`, `後方ステップ`, `パリィドライブラッシュ`. |
| ダメージが違う | 16 | Modern側ではSP/簡易入力の低下ダメージが表示され、括弧内に通常入力時の値が併記される場合がある。 |
| 発生 / 持続 / 硬直 / ヒット / ガード / キャンセル / ゲージ / 属性が違う | 0 | この時点の公式表では、共通する技名の項目に差分なし。 |

共通項目の入力差分は、おおむね次のパターン。

| 種類 | Classicの例 | Modernの例 |
|---|---|---|
| 通常技ボタン | `弱P`, `中K`, `強P` など | `弱`, `中`, `強`, `AUTO+中` など |
| 必殺技 | `↓↘→ + 弱P` のようなコマンド+パンチ/キック | `N+SP`, `方向+SP`, Modernボタンなど |
| スーパーアーツ | 2回転系コマンド+パンチ/キック | SPショートカット入力。括弧内に通常入力が併記される場合がある |
| ドライブシステム | `強P+強K`, `中P+中K` などの複数ボタン | `DI`, パリィなどのModern専用ボタン |
| 移動だけの項目 | `→→` など | 変わらない3項目では同じ方向入力 |

入力差分の例:

- `立ち弱P（ノーシ）`: Classicは `弱P`、Modernは `弱`。
- `弱 ストリボーグ`: Classicは `↓↘→ + 弱P`、Modernは `ニュートラル + SP`。
- `SA1 チェルノボーグ`: Classicは `↓↘→↓↘→ + P`、Modernは `ニュートラルまたは→ + SP / 強`。
- `ドライブインパクト（エスパーダ）`: Classicは `強P+強K`、Modernは `DI`。
- `前方ステップ`: どちらも `→→`。

ダメージ差分がある共通項目:

| 技 | Classicダメージ | Modernダメージ |
|---|---:|---:|
| アブニマーチ | 1800 | 1440 |
| OD アブニマーチ | 2600 | 2080 |
| アムネジア | 800 | 640 |
| OD アムネジア | 1200 | 960 |
| 弱 ストリボーグ | 1000 | 800 |
| OD ストリボーグ | 1000 | 800 |
| 弱 トリグラフ | 800 | 640 (800) |
| 中 トリグラフ | 800 | 640 (800) |
| 強 トリグラフ | 800 | 640 (800) |
| OD 弱 トリグラフ | 1000 | 800 (1000) |
| OD 中 トリグラフ | 1000 | 800 (1000) |
| OD 強 トリグラフ | 1000 | 800 (1000) |
| SA1 チェルノボーグ | 2000 | 1600 (2000) |
| SA2 ラヴーシュカ | 2000 | 1600 (2000) |
| SA3 ザプリェット | 4000 | 3200 (4000) |
| CA ザプリェット | 4500 | 3600 (4500) |

## 注意

- この回答では、公式フレームデータ表に載っている1項目を単位として数えている。
  技分類全体への一般論ではない。
- 入力比較は、公式表の「技名」と、公式表から取得した入力表示の完全一致に
  基づいている。本文中の矢印表記は、読みやすさのために公式表の入力表示を
  置き換えたもの。
- 「ダメージ以外のフレーム系値に差分なし」は、2026-05-26時点のJP公式表にある
  共通技名の項目に限った話。Classic/Modernのあらゆる実戦状況が同じだと
  いう意味ではない。
- この回答は2026-05-26時点の公式表に基づくため、後のCapcom更新で項目数や
  入力が変わる可能性がある。
