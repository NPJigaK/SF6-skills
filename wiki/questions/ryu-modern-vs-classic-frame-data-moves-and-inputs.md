# Question: Ryuのモダンとクラシックで、フレームデータ上の技数や入力はどう違いますか？

## 短い答え

RyuのModern版も、Classicと同じ技表をそのまま別入力にしたものではない。
公式Ryuフレームデータでは、Classicは75項目、Modernは69項目で、Modernの
ほうが6項目少ない。

ただし、公式表の「技名」を完全一致で比べると、Classicにだけある技名は7つ、
Modernにだけある技名は1つになる。これはClassicの `しゃがみ強K（回転足払い）`
に対して、Modern側では `回転足払い` という別名の項目があるため。

Classicにだけある技名は次の7つ。

| カテゴリー | Classicのみの技名 | Classic入力 |
|---|---|---|
| 通常技 | 立ち中K（横蹴り） | `中K` |
| 通常技 | 立ち強K（後ろ回し蹴り） | `強K` |
| 通常技 | しゃがみ強K（回転足払い） | `↓ + 強K` |
| 特殊技 | 上げ突き | `← + 強P` |
| 必殺技 | 弱 竜巻旋風脚 | `↓↙← + 弱K` |
| 必殺技 | 強 竜巻旋風脚 | `↓↙← + 強K` |
| 必殺技 | 強 上段足刀蹴り | `↓↘→ + 強K` |

Modernにだけある技名は次の1つ。

| カテゴリー | Modernのみの技名 | Modern入力 |
|---|---|---|
| 特殊技 | 回転足払い | `↘ + 強` |

入力表示は大半が異なる。公式表の「技名」が同じ68項目のうち、入力表示が
完全に同じなのは `前方ステップ`、`後方ステップ`、`パリィドライブラッシュ`
の3項目だけで、残り65項目はModernとClassicで入力表示が異なる。

補足すると、技名が同じ68項目では、入力とダメージ以外のフレーム系の値は
この公式表では一致している。一部のModern入力ではダメージが下がる。

## 根拠

この回答は、2026-05-27に取得し、人間レビューでacceptedになったCapcom公式
Ryuフレームデータを根拠にしている。Classic/Modernの表を項目単位で比較し、
技名、入力表示、動作フレーム、硬直差、キャンセル、ダメージ、ゲージ増減、
属性、備考の差分を確認した。

Modern側の補足説明には、SPボタンで発動した場合のダメージが80%になるという
説明がある。そのため、Modernの一部項目で `560 (700)` のように低下ダメージと
通常入力時の値が併記されている。

## 比較

カテゴリー別の項目数は次の通り。

| カテゴリー | Classic | Modern | 差 |
|---|---:|---:|---:|
| 通常技 | 18 | 15 | -3 |
| 特殊技 | 8 | 8 | 0 |
| 必殺技 | 27 | 24 | -3 |
| スーパーアーツ | 10 | 10 | 0 |
| 通常投げ | 2 | 2 | 0 |
| 共通システム | 10 | 10 | 0 |
| 合計 | 75 | 69 | -6 |

公式表の「技名」が完全に同じ項目で比較すると、共通項目は68、Classicのみは7、
Modernのみは1。

| 差分の種類 | 件数 | 備考 |
|---|---:|---|
| 入力表示が違う | 65 | 多くのClassic入力が、Modernボタン、SP、AUTO、方向+SPなどに置き換わる。 |
| 入力表示が同じ | 3 | `前方ステップ`, `後方ステップ`, `パリィドライブラッシュ`. |
| ダメージが違う | 22 | Modern側ではSP/簡易入力の低下ダメージが表示され、括弧内に通常入力時の値が併記される場合がある。 |
| 発生 / 持続 / 硬直 / ヒット / ガード / キャンセル / ゲージ / 属性 / 備考が違う | 0 | この時点の公式表では、同じ技名の項目に差分なし。 |

入力差分の例:

- `立ち弱P（ジャブ）`: Classicは `弱P`、Modernは `弱`。
- `弱 波動拳`: Classicは `↓↘→ + 弱P`、Modernは `↓↘→ + 弱`。
- `強 昇龍拳`: Classicは `→↓↘ + 強P`、Modernは `→ + SP`。
- `SA1 真空波動拳`: Classicは `↓↘→↓↘→ + P`、Modernは `ニュートラル + SP / 強`。
- `前方ステップ`: どちらも `→→`。

## 注意

- この回答では、公式フレームデータ表に載っている1項目を単位として数えている。
  技分類全体への一般論ではない。
- 「技名が同じ項目」は、公式表の「技名」が完全一致するものだけを数えている。
  `しゃがみ強K（回転足払い）` と `回転足払い` のように、実質的には対応しそう
  でも技名が違うものは別項目として扱っている。
- 本文中の矢印表記は、読みやすさのために公式表の入力表示を置き換えたもの。
- 「ダメージ以外のフレーム系値に差分なし」は、2026-05-27時点のRyu公式表に
  ある同名項目に限った話。Classic/Modernのあらゆる実戦状況が同じだという
  意味ではない。
- この回答は2026-05-27時点の公式表に基づくため、後のCapcom更新で項目数や
  入力が変わる可能性がある。
