# 仕込み

## 要約

仕込みは、cancel 可能な通常技の motion 中に次の行動を入力しておき、条件が満たされた時だけ次行動を出す technique。Capcom eSports BASE は、置き技が届いた時だけ cancel action が出る形を中心に説明している。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/capcom-esports-base-terms]] `word-16` | 届かない間合いで cancel 可能な通常技を振り、motion 中に必殺技 command などを入れておくことで、通常技が届いた場合だけ次行動が出る。歩き guard などで間合いを調整して仕込み漏れを誘う対策がある。 | high |

## 意味差分

- `option select` と完全同義とは限らないため、slug は `shikomi` にした。今後の source 追加時に buffer / option select / whiff-cancel 系の意味差分を分ける。
