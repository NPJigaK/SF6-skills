# Blockstring

## 要約

Blockstring は、相手に guard させながら攻めを継続する sequence。SuperCombo Offense page は攻め側の sequence として、SuperCombo Defense page は true blockstring 中の auto guard と defensive block behavior として説明する。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/supercombo-street-fighter-6-offense]] | Blockstring は、guard している相手を lock down し、確定 damage ではなく opening を作るために使う sequence。 | medium |
| [[sources/supercombo-street-fighter-6-offense]] | Invincible moves でも抜けられない blockstring は true / airtight blockstring と呼ばれる。 | medium |
| [[sources/supercombo-street-fighter-6-offense]] | SF6 では、多くの blockstrings は chained light normals または Drive Rush 強化 starter に関わり、medium / heavy normals や special moves だけで相手を簡単に lock down できる character は少ないと説明される。 | medium |
| [[sources/supercombo-street-fighter-6-defense]] | True blockstring 中の defender は mid / overhead を auto guard するが、Low は low block が必要で、crouch block 中の overhead は auto-guard されない。 | medium |

## 意味差分

- [[concepts/terms/lock]] は SuperCombo HUD page の Drive Impact corner block cue / true blockstring cue を扱う。この page は攻めの sequence としての blockstring を扱う。
- [[concepts/terms/blocking]] は defender 側の guard behavior を扱う。
- True / airtight blockstring と、4f normal で割り込みにくい一般的な pressure sequence は区別する。
- 個別 blockstring の成立は frame advantage、pushback、cancel timing、Drive Rush、defender options に依存する。
