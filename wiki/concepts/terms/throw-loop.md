# Throw Loop

## 要約

Throw Loop は、投げ後の frame advantage と距離が足りていて、相手の wake-up にもう一度 meaty throw を迫れる状況を指す community term。SuperCombo Offense page は、多くの場合 corner 限定で、midscreen Back Rise には届きにくいが、Punish Counter throw の Hard Knockdown や Drive Rush によって例外が生まれると説明している。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/supercombo-street-fighter-6-offense]] | Throw loop は、投げ後に十分な frame advantage があり、相手 wake-up に次の meaty throw を試せる状況を指す community term。 | medium |
| [[sources/supercombo-street-fighter-6-offense]] | 通常は corner で成立し、midscreen では Back Rise で投げ間合い外へ逃げられやすい。Punish Counter throw は Hard Knockdown で Back Rise を無効化するため例外になる。 | medium |
| [[sources/supercombo-street-fighter-6-offense]] | Character ごとに walk、forward dash、Drive Rush、whiff normal frame-kill などの setup が異なる。 | medium |

| Zangief | `3LPLK` は midscreen Back Rise にも genuine throw loop を与えるが、Drive Rush と 2f timing window が必要と説明される。 |

## 意味差分

- Throw loop は公式用語ではなく、SuperCombo Offense page 自身も community term として説明している。
- Character-specific notes は patch 変更や lab 更新で stale になりやすい。現在値の断定には公式 frame-data、patch note、実機検証を併用する。
- [[concepts/terms/oki]] や [[concepts/terms/meaty]] より狭く、投げを継続的に迫れる状況に絞る。
