# 無敵技

## 要約

無敵技は、一定時間相手の攻撃を受け付けない性質を持つ技。Capcom eSports BASE は不利状況を切り返す high-risk な行動として説明し、SuperCombo Defense page は wake-up / defensive reversal の文脈で true invincibility と armor の違いを補う。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/capcom-esports-base-terms]] `word-12` | 無敵時間を持つ技。不利状況を切り返せるが、外れると大きな隙になる。完全無敵だけでなく、打撃無敵、投げ無敵、飛び道具無敵など属性別の無敵もある。 | high |
| [[sources/supercombo-street-fighter-6-defense]] | Reversal で特に有用なのは invincibility を持つ技で、meaty attack に勝てる。Source は true startup invincibility を持ち得るのは OD Special Moves と Supers だけだが、すべてがそうではないと説明する。 | medium |

## 意味差分

- 個別技がどの属性に何 frame 無敵かは frame-data source で確認する必要がある。
- [[concepts/terms/armor]] は攻撃を吸収する property であり、true invincibility とは違う。
