# 対空

## 要約

対空は、jump などで空中にいる相手を迎撃する攻撃や選択肢の総称。Capcom eSports BASE は、対空必殺技、通常技対空、空対空、parry 対空を例として挙げている。SuperCombo Defense page は、air invincibility、air-to-air、walk-under、low profile などの defensive counterplay を補う。

## 意味差分

- 現時点では一般戦術語として扱う。個別 character の対空技、air invincibility、frame は各 character / frame-data source を参照する。
- Source の例示は community tactical context であり、どの攻撃が airborne 扱いかは個別 move property を確認する。
