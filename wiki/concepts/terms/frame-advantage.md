# 有利フレーム

## 要約

有利フレームは、hit / guard 後に自分が相手より先に動ける時間。Capcom eSports BASE は 1 frame = 1/60 秒として説明し、Drive Rush からの通常技は有利 frame が増えるため強力だとしている。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/capcom-esports-base-terms]] `word-13` | 1 frame は 1/60 秒。有利 frame は hit / guard 後に自分が先に動き出せる時間で、不利 frame は相手が先に動ける時間。Drive Rush 通常技では有利 frame が増える。 | high |
| [[sources/supercombo-street-fighter-6-glossary]] | Hit / block advantage は、攻撃後にどちらがどれだけ先に動けるかを示す frame-data vocabulary として説明されている。 | medium |

## 意味差分

- Capcom eSports BASE は観戦向けの日本語説明、SuperCombo は frame-data 用語としての community explanation。大きな意味差分は未確認。
