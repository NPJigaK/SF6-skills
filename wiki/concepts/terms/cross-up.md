# めくり

## 要約

めくりは、主に jump attack で相手の背中側を狙い、guard direction を分かりにくくする崩し。Capcom eSports BASE は、成功時に `CROSS UP` 表示が出ること、Drive Parry が表裏両対応の対抗策になり得ることを説明している。SuperCombo HUD page は表示 cue を、SuperCombo Defense page は block direction と corner interaction を補う。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/capcom-esports-base-terms]] `word-04` | 相手の背中付近を狙い、逆方向 guard を要求する jump attack 系の崩し。成功時は `CROSS UP` 表示が出る。Drive Parry で両対応できる場合がある。 | high |
| [[sources/supercombo-street-fighter-6-hud]] | `Cross-up` icon は、攻撃が相手を crossed over し、相手が向いていた方向とは逆方向の block を強制する時に表示される。 | medium |
| [[sources/supercombo-street-fighter-6-defense]] | Cross-up は通常、相手の下を歩くように forward guard する必要がある。Fully cornered opponent には cross-up が connect せず、どちら方向でも block されると説明する。 | medium |

## 意味差分

- 現時点では jump attack を中心にした崩しとして扱う。表裏 setplay との関係は追加 source が必要。
- SuperCombo HUD page は display cue の説明として使い、対策や system detail は公式 source を優先する。
- Corner の cross-up interaction は spacing と hurtbox に依存する community tactical claim として扱う。
