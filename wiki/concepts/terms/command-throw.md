# コマ投げ

## 要約

コマ投げは、特定 character が持つ command throw / special throw。Capcom eSports BASE は、通常投げと異なり throw tech できない点、大きな reward と whiff / read された時の大きな risk を説明している。

## Source claims

| Source | Claim | Confidence |
|---|---|---|
| [[sources/capcom-esports-base-terms]] `word-05` | command throw は throw tech できず damage も大きいが、vertical jump や backstep などで避けられると大きな反撃を受ける。 | high |

## 意味差分

- 現時点では一般用語として扱う。個別 character の command throw 性能は frame-data source が必要。
