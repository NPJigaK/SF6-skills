# セットプレイ

## 要約

セットプレイは、相手の行動不能時間を利用し、あらかじめ pattern 化した起き攻めや攻め継続を仕掛けること。Capcom eSports BASE は、down 後や浮かせた後に使われ、空振りによる frame kill で timing を調整することもあると説明している。

## Source claims

## 意味差分

- 現時点では起き攻め寄りの説明として扱う。projectile / summon / install など character-specific setplay は追加 source が必要。
