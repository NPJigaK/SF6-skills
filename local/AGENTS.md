# Local Trial Workspace

This workspace is for personal trials of distributed skills from the parent repo.

Public skill source lives in `../skills/`.
Shared maintainer workflows live in `../maintainer-skills/`.

Generated media, transcripts, outputs, and personal helper skills belong under ignored paths in this workspace.

If discovery links are missing, run `pwsh -File ../scripts/dev/bootstrap-local-trial-workspace.ps1`.

When analyzing SF6 video from a YouTube link, keep downloaded media and generated outputs under ignored local paths.
