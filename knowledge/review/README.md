# Knowledge Review

This directory holds canonical review tracking for material that is not yet accepted curated knowledge or final answer evidence.

Review entries should be split into structured claim or review artifacts before acceptance.

## Subdirectories

- `unresolved/`: claims or source summaries that lack enough evidence for acceptance.
- `contested/`: conflicting claims or interpretations that need reviewer resolution.
- `current-fact-candidates/`: patch-sensitive exact-current candidates that need current-fact workflow handling before they can influence public answers.

Review material must not feed generated knowledge references unless it is promoted into `knowledge/curated/`.
